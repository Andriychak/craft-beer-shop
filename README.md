# Craft Beer Shop - Інтернет-магазин крафтового пива

Повнофункціональна web-аплікація для управління інтернет-магазином крафтового пива з фронтенд та бекенд частинами.

## Особливості

✅ **Управління товарами:**
- Створення нових елементів
- Редагування існуючих елементів
- Видалення елементів
- Сортування за параметрами (назва, ціна, крепкість)
- Пошук товарів за назвою, описом та пивоварнею

✅ **Робота з корзиною:**
- Додавання товарів в корзину
- Видалення товарів з корзини
- Оновлення кількості товарів
- Очищення корзини
- Підрахунок суми покупок

✅ **Технологічний стек:**
- **Frontend:** HTML5, CSS3, JavaScript (ES6+) з принципами ООП
- **Backend:** Node.js + Express
- **Database:** SQLite3
- **Design:** Dark theme, Mobile First, Responsive

## Архітектура

### Frontend (JavaScript)

Код організований за принципами ООП з наступними класами:

- **`APIService`** - управління API запитами до сервера
- **`Product`** - модель товару
- **`CartItem`** - модель товару в корзині
- **`Cart`** - управління корзиною
- **`UIManager`** - управління інтерфейсом
- **`App`** - головний клас застосунку

### Backend (Node.js + Express)

API endpoints:
- `GET /api/products` - отримати всі товари (з пошуком і сортуванням)
- `GET /api/products/:id` - отримати конкретний товар
- `POST /api/products` - створити новий товар
- `PUT /api/products/:id` - оновити товар
- `DELETE /api/products/:id` - видалити товар
- `GET /api/cart` - отримати корзину
- `POST /api/cart` - додати товар в корзину
- `DELETE /api/cart/:id` - видалити товар з корзини
- `PUT /api/cart/:id` - оновити кількість товару
- `DELETE /api/cart` - очистити корзину

## Встановлення та запуск

### Вимоги
- Node.js (v14+)
- npm

### Кроки встановлення

1. **Встановіть залежності:**
```bash
npm install
```

2. **Запустіть сервер:**
```bash
npm start
```

Сервер запуститься на `http://localhost:3000`

3. **Відкрийте браузер:**
```
http://localhost:3000
```

### Для розробки (з auto-reload)

```bash
npm install --save-dev nodemon
npm run dev
```

## Використання

### Каталог товарів
- Переглядайте всі товари на головній сторінці
- Пошук товарів за назвою, описом або пивоварнею
- Сортуйте товари за: назвою, ціною, крепкістю, датою додавання

### Додавання товару
1. Натисніть "Додати товар" в меню
2. Заповніть форму з деталями пива:
   - Назва (обов'язково)
   - Пивоварня
   - Ціна (обов'язково)
   - Крепкість (%)
   - Об'єм (мл)
   - Посилання на зображення
   - Опис
3. Натисніть "Додати товар"

### Редагування товару
1. На карточці товару натисніть "Редагувати"
2. Змініть потрібні параметри
3. Натисніть "Зберегти зміни"

### Видалення товару
1. На карточці товару натисніть "Видалити"
2. Підтвердіть видалення в модальному вікні

### Робота з корзиною
1. Натисніть "До корзини" на карточці товару
2. Перейдіть в корзину через меню
3. Змініть кількість товарів або видаліть їх
4. Перегляньте загальну суму покупок

## Бази даних

SQLite база даних створюється автоматично при першому запуску сервера.

### Таблиці

**products**
```sql
CREATE TABLE products (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  description TEXT,
  price REAL NOT NULL,
  alcohol REAL,
  volume INTEGER,
  brewery TEXT,
  image TEXT,
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
)
```

**cart**
```sql
CREATE TABLE cart (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  product_id INTEGER NOT NULL,
  quantity INTEGER NOT NULL,
  FOREIGN KEY (product_id) REFERENCES products(id) ON DELETE CASCADE
)
```
